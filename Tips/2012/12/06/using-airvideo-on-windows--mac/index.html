Using AirVideo on Windows / Mac

AirVideo 제작사에서는 데스크탑 운영체제용 AirVideo 플레이어를 제공하고 있지 않다. 동영상 스트리밍을 위한 다양한 솔루션들이 있겠지만, 이미 AirVideo 서버를 운영하고 있다면 이를 최대한 활용해보자.

Windows

다운로드: http://dl.dropbox.com/u/10205542/AirVideoClient_win32_20111218.zip

Media Player 혹은 QuickTime Player 를 이용해 동영상을 재생한다. 다른 재생 프로그램을 사용하려면 동영상 파일 URL을 복사해 붙이면 된다.

Mac OS X

다운로드: http://dl.dropbox.com/u/6713110/MacAirVideoClient/Site/Home.html

VLC Player를 이용해 동영상을 재생한다. 서버 주소를 등록하고 동영상 파일을 선택하면, VLC 가 실행되어 동영상을 재생한다.
